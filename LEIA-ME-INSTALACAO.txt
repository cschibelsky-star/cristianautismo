CRISTIAN PELO AUTISMO - V11 AJUSTADA

1) Como publicar no cPanel / HostGator
Extraia o conteúdo do ZIP direto na raiz do domínio:

public_html/index.html
public_html/style.css
public_html/script.js
public_html/assets/
public_html/campanha/
public_html/campanha/admin/
public_html/leis-e-direitos/

Não deixe dentro de uma pasta interna como public_html/cristianautismo_site_premium_v11/.

2) Correção visual
Esta versão ajusta a primeira dobra da Home para evitar sobreposição entre o banner/logo e o mockup principal.
Os mockups continuam como ícones de navegação 01 a 06.

3) Painel admin da campanha
Acesso:
/campanha/admin/

Senha inicial provisória:
Cristian@2026!

IMPORTANTE: troque esta senha depois de publicar.
O painel usa sessão PHP.

4) Como trocar a senha do admin
Abra o arquivo:
/campanha/config.php

No terminal, gere um novo hash com:
php -r 'echo password_hash("SUA_NOVA_SENHA", PASSWORD_DEFAULT), PHP_EOL;'

Cole o hash gerado no lugar do valor de CRISTIAN_ADMIN_PASSWORD_HASH.

5) Dados dos apoios
Os apoios ficam em:
/campanha/data/apoios.json

A pasta data possui .htaccess para bloquear acesso direto.
